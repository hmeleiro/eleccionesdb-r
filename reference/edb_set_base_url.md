# Set the EleccionesDB API base URL

Set the EleccionesDB API base URL

## Usage

``` r
edb_set_base_url(url)
```

## Arguments

- url:

  Character string with the base URL (e.g.
  `"https://api.spainelectoralproject.com/"`).

## Value

The previous base URL, invisibly.

## Examples

``` r
if (FALSE) { # \dontrun{
edb_set_base_url("http://my-server:8000")
} # }
```
